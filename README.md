# TB_Smith_Charts
Réalisation d'une application native pour visualiser et utiliser une abacque de smith dans le contexte de l'enseignement à l'HEIG-VD.


## Rapport Typst

Le rapport de ce projet est élaboré en utilisant le langage de mise en page Typst ainsi qu'un template développé par Christophe Roulin et Samuel Roland : https://github.com/DACC4/HEIG-VD-typst-template-for-TB

Le readme servera de note d'accompagnement pour le projet, du moins pour le moment.
